Um código de fonte está em conformidade com um modelo de arquitectura, quando se adere plenamente a um conjunto de restrições de arquitectura. Uma restrição permite negar referências entre classes por padrão.

Você pode, por exemplo, usar essa regra para:

  • proibir o acesso às classes **.web.** from **.dao.**
  • proibir o acesso a jjava.util.Vector, java.util.Hashtable and java.util.Enumeration a partir quaisquer classes
  • proibir o acesso a java.sql.** a partir das classes em **.ui.** e **.web.**
