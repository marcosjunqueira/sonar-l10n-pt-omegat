Aqui estão as principais razões pelas quais código comentado cheira mal:

  • Ele sempre levanta mais perguntas do que dá respostas
  • Todo mundo vai esquecer rapidamente o quão relevante é o código comentado
  • É uma distração ao ler o código, interrompendo o fluxo de leitura
  • É um mecanismo SCM ruim: Subversion, CVS e Git são realmente mais confiáveis!
  • O simples fato de entender por que o código foi comentado em primeiro lugar pode tomar muito tempo
